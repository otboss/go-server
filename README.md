<h1>Go Server</h1>

<h2>A simple and performant http server</h2>

<h3>Steps:</h3>
<ol>
    <li>
        <span>Change go path</span>
        <pre>
            export GOPATH=$(pwd)
        </pre>
    </li>
    <li>
        <span>Run server</span>
        <pre>
            go run main.go
        </pre>
    </li>
</ol>
